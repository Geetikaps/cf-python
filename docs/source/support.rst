.. _Contributing:

**Contributing**
----------------

Contributions to cf-python are welcome and should be raised as issues
in the GitHub issue tracker:
https://github.com/NCAS-CMS/cf-python/issues, prior to submitting a
pull request containing the new code made from a fork of the cf-python
GitHub code repository: https://github.com/NCAS-CMS/cf-python.
